# Exploring Trends in American Baby Names (SQL)

End-to-end SQL analysis completed in DataCamp of 100+ years of U.S. baby name data, exploring long-term trends, ranking dynamics, and classic vs. trendy name patterns.

## Key Questions Answered

1. List the first five names in alphabetical order and find out if each name is "Classic" or "Trendy."
2. Identify the top 20 male names overall and analyze how the name Paul ranks.
3. Identify which female names appeared in both 1920 and 2020.

## Files

- `analysis.sql` — final, documented SQL queries
- `exploration.ipynb` — DataCamp-exported notebook (supporting work)
